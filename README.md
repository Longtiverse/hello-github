# My First GitHub Project 🚀

这是我的第一个 GitHub 学习项目，记录了我的通关历程！

## 🛠 已掌握的技能
* **基础操作**：Git 安装、配置及三板斧（Add, Commit, Push）。
* **版本控制**：分支管理（Branch）与冲突解决。
* **进阶技巧**：使用 `.gitignore` 过滤文件，处理嵌套仓库。

## 📖 学习心得
> “Git 并不是为了记录代码，而是为了记录思维的演进。” —— 我自己说的

## 运行本项目
1. 克隆到本地：
   `git clone https://github.com/Longtiverse/hello-github.git`
2. 查看笔记：
   直接打开 `note.txt` 即可。
